# Stock Portfolio Tracker

A Python-based mini finance application that calculates investment value based on selected stocks and quantity entered by the user.

## Features
- User input for stock name and quantity
- Predefined stock price system
- Automatic investment calculation
- Displays portfolio summary

## Technologies Used
- Python
- Dictionaries
- User Input Handling
- Arithmetic Operations

## Execution
Run the stock.py file, enter stock details, and view the total investment amount instantly.

## Learning Outcome
This project enhanced my understanding of dictionaries, calculations, and real-world Python applications.
